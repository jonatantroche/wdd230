Fonts

Poppins Regular 400

Roboto Regular 400

Applications

I created a contrast between the title and the main text with the bold of the Poppins font and a light easy-to-read Roboto font.

Uses

It is intended to use Poppins for all titles, subtitle,s and text that you want to highlight, these are normally in bold, on the other hand, any other text will use Roboto as the main font
